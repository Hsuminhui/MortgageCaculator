# 房貸試算器

專案簡介:此專案為一個房貸試算器，能夠計算出每月應繳金額、首期本金、首期利息、總利息支出、總還款金額。

執行說明:

1.輸入房屋總價，此專案將幣值設定為新台幣(NT$)，當輸入的值為空or負數值，會跳出錯誤視窗以提醒使用者。

2.使用者自行選擇輸入自備款比例(%)或自備款金額，當選擇自備款比例(%)時，輸入值必須介於0~99.99之間，否則將跳出錯誤視窗;
  當選擇自備款金額時，輸入值須為正數且小於房價的價格。

3.輸入貸款利率(%)，需大於0。

4.輸入貸款年限，需大於0。

5.選擇寬限期，選擇之寬限期需小於貸款年限，亦不能等於貸款年限(等於在還款年限間只支付利息，本金依舊沒有減少)，否則跳出錯誤視窗。

6.按下"開始計算"按鈕後，能夠自動計算每月應繳金額、首期本金、首期利息、總利息支出、總還款金額，並呈現在輸出視窗中。

7.按下"清除"按鈕後，輸入視窗及輸出視窗會清除所有的值。

其他功能:

1.本專案設置無滑鼠操作，能使用tab、enter進行輸入及計算。

2.提供專業名詞解釋及相關的計算公式，為可滑動視窗。

3.當輸入金額時，超過千位會有千位標記，供使用者閱讀便捷。
